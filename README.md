## Getting started

### Requirements

* [Docker](https://docker.com/)
* [Docker Compose](http://docs.docker.com/compose/)

### Instructions

```sh
# Clone the repository (using hub)
$ git clone git@github.com:Dareen/yii2php.git
$ cd yii2php

# Run using make
$ make docker
```
